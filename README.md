# Админ-панель для создания и управления учебными курсами платформы CunaEdu.

**Тема проекта:** Система управления образовательным контентом (LCMS) для создания курсов с иерархической структурой.

Проект использует **React** и **Tailwind CSS** (версия 3.x).

**Ссылка на сайт:** https://yanokavi.github.io/AdminPanelCunaEdu/
